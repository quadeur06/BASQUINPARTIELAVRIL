2) shift+ clique droit pour ouvrir powershell
3) git init pour initialiser git sur le dossier du partiel
4 ) ajouter a mon fichier .git la liaison entre github et le dossier avec cette ligne "  git remote add origin https://github.com/quadeur06/BAS

5 ) creer un fichier .gitignore dans le meme dossier et ajouter .vs/ pour eviter une erreur
6) ajouter tout les fichiers avec git add --all
7 ) faire un commit avec git commit -m " premier commit"

8 ) envoyer avec git push -u origin master

9 ) ajout d'une branche avec  git checkout -b "dev"
